# Superstore Sales Performance Dashboard (2014–2017)

## Project Overview
This project analyzes Superstore sales data from 2014 to 2017 using Google Sheets. The dashboard provides insights into sales performance across regions, categories, customer segments, and states.

## Tools Used
- Google Sheets
- Pivot Tables
- Charts & Visualizations
- Data Cleaning
- Dashboard Design

## Key Metrics
- Total Sales: 2.30 Million
- Total Orders: 9,995
- Total Customers: 9,994
- Top State: California

## Key Insights
- West region generated the highest sales.
- Consumer segment contributed the most revenue.
- Technology category recorded the highest sales.
- California was the top-performing state.
- Technology category generated the highest profit.
- Sales showed strong growth from 2015 to 2017.

## Dashboard Components
1. Region-wise Sales Analysis
2. Category-wise Sales Analysis
3. Segment-wise Sales Analysis
4. Top 10 States by Sales
5. Sales Trend (2014–2017)
6. Category-wise Profit Analysis
